Layer 2B: Expert Evaluation — ส่วนที่ทำให้งานแข็งแรงขึ้นมาก
ทำไมต้องมี
Simulation รอบเดียวไม่มี statistical power → ต้องมี คนที่รู้เรื่องฟุตบอล มาประเมินว่าผลลัพธ์ของระบบสมเหตุสมผลไหม
วิธีทำ
หาผู้เชี่ยวชาญ 3-5 คน (ไม่ต้องเป็น scout จริง — แค่คนที่ดูบอลจริงจังหรือเล่น FM บ่อย)

ภาพรวม Evaluation ทั้งหมด
Evaluation
├── Layer 1: ML Model (quantitative — มีแล้ว)
│   ├── 5-fold CV accuracy / R²
│   ├── Paired t-test (RF vs XGBoost)
│   └── SHAP feature importance
│
└── Layer 2: System (ต้องเพิ่ม)
    ├── 2A: Simulation Case Study (quantitative)
    │   └── FM2024 Season 1 vs Season 2 metrics
    │
    └── 2B: Expert Evaluation (qualitative + quantitative)
        └── Likert-scale questionnaire (3-5 experts)
ทำไม framework นี้แข็งแรง

- Layer 1 ตอบว่า "model แม่นไหม"
- Layer 2A ตอบว่า "ใช้จริงแล้วทีมดีขึ้นไหม"
- Layer 2B ตอบว่า "คนที่รู้เรื่องฟุตบอลเห็นด้วยกับผลลัพธ์ไหม"
  ทั้ง 3 ชั้นเสริมกัน — ถ้า simulation รอบเดียวอาจารย์บอกว่า weak ก็มี expert evaluation มา backup ได้ครับ
